BrowserStack Live allows you to test websites on real browsers and operating systems. With **Requestly**, you can modify network requests, mock APIs, redirect URLs, inject scripts, and debug your application directly inside your live sessions.

You can use Requestly in BrowserStack Live in two ways:

1. **Preload the Requestly extension** (recommended)  
2. **Manually install the extension** inside the browser session


## Preloading the Requestly Extension (Recommended)

BrowserStack allows certain extensions to be preloaded so they are automatically available in every new session. This is the fastest and most reliable way to use Requestly in Live.

### Steps to Preload Requestly

<Steps>
<Step title="Log in to BrowserStack Live">
**Log in** to your BrowserStack **[Live dashboard](https://live.browserstack.com/dashboard)**.
</Step>

<Step title="Open Extensions">
Click the **Extensions** button next to the bookmark icon in the URL bar.
![Extensions Button](/images/how-to-use-requestly-with-browserstack-live/extensions-button.webp)
</Step>

<Step title="Enable Requestly">
The **Extensions window** opens with the **Requestly extension already added** by default. Enable the extension using the toggle switch.
![Enable Requestly Extension](/images/how-to-use-requestly-with-browserstack-live/toggle-extension.webp)
</Step>

<Step title="Start Live Session">
Start a Live session by selecting a supported **OS** and **browser combination**. Requestly loads automatically inside the browser.
![Requestly in BrowserStack Live](/images/how-to-use-requestly-with-browserstack-live/start-a-live-session.webp)
</Step>
</Steps>


### Supported OS & Browsers for Preloading

Preloading is supported only in **desktop sessions**.

| Operating System | Supported Browsers                         |
|------------------|---------------------------------------------|
| **Windows**      | Chrome (up to version 142), Edge            |
| **macOS**        | Chrome (up to version 142), Edge            |

> **Note:** Firefox does not support preloading extensions. You can still install Requestly manually during your session.


## Manually Installing Requestly in BrowserStack Live

If your browser or session does not support preloading, you can manually install Requestly inside the Live session.


### Chrome / Edge

<Steps>
<Step title="Start Live Session">
Start a Live session using **Chrome** or **Edge**.
</Step>

<Step title="Open Chrome Web Store">
Open the **Chrome Web Store**.
</Step>

<Step title="Search for Requestly">
Search for **Requestly**.
</Step>

<Step title="Install Extension">
Click **Add to Chrome** / **Add to Edge**.
</Step>

<Step title="Extension Ready">
The extension is now available in the browser.
</Step>
</Steps>


### Firefox

For Firefox users, Requestly can be installed through the Requestly download page.

<Steps>
<Step title="Start Live Session">
Start a Live session using **Firefox**.
</Step>

<Step title="Visit Download Page">
Visit the Requestly download page: **https://requestly.in/download**
</Step>

<Step title="Download Extension">
Download the **Requestly Firefox (.xpi) extension**.
</Step>

<Step title="Approve Installation">
When prompted, approve the installation.
</Step>

<Step title="Extension Ready">
Requestly is now available inside the browser.
</Step>
</Steps>
