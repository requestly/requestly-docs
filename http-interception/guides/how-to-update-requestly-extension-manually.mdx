If Chrome hasn't auto-updated Requestly Interceptor yet, here's how to force the latest version instantly.

By default, Chrome automatically updates extensions when a new version is published to the Chrome Web Store. However, this process can take anywhere from a few hours to a couple of days.

If you want to update the Requestly Interceptor extension immediately, follow these steps:

<Steps>
  <Step title="Open Extensions Page">
    Go to your Chrome browser and type `chrome://extensions/` in the address bar. Hit Enter.
  </Step>

  <Step title="Enable Developer Mode">
    In the top-right corner of the Extensions page, toggle on **Developer mode**.

    <img src="/images/how-to-update-requestly-extension-manually/c35f387a-5e15-4125-b013-589d18894d1b.png" align="center" fullwidth="false" />
  </Step>

  <Step title="Find Requestly Interceptor">
    Use the search bar at the top to search for “Requestly Interceptor.”
  </Step>

  <Step title="Click the Update Button">
    On the Requestly Interceptor extension card, click the **Update** button. Chrome will fetch and install the latest version.

    <img src="/images/how-to-update-requestly-extension-manually/accc0831-b8d6-42df-a9ba-44ef44d0d595.png" align="center" fullwidth="false" />
  </Step>
</Steps>
