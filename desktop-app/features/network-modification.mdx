# Steps

1. Right Click on any Network Request in the Requestly Network Interceptor

2/images/network-modification/360df924-9811-4b40-8200-3ff4773da1dc.png" align="center" />

   Select the Rule you want to create.

3. Fill in the details of modification in the Rule Editor Modal.

<img src="https://imagedelivery.net/IEMzXmjRvW0g933AN5ejrA/wwwnotionso-image-prod-files-secures3us-west-2amazonawscom-8e04dd23-74cd-4c44-8158-4238eac359d1-dd234d3f-ae4c-4ee4-8dd4-9bddbe01755b-ss-2_18png/public" align="left" />

4. Click `Create Rule` on the top-right in the Rule Editor Modal.

# Testing Modification

Now when the same request is triggered again, the request will be highlighted. This indicates that the Requestly Rule is applied on the request.
