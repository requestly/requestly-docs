Incase the app has shut down suddenly while you had the system wide proxy set, here are the steps to make sure that system wide proxy settings have been disabled on MacOS

Â 

The steps are slightly different for before and after macOS 13

### **macOS 13 Ventura or later**

1. Open System Preferences

2) Type Proxies in Search

3. Select Proxies

4) Switch off Web Proxy (HTTP) and Secure Web Proxy (HTTPS) to disable the HTTP and HTTPS proxies

5. Press OK

### **Before macOS 13 Ventura**

1. Open System Preferences

2) Go to Network

3. Select Advanced

4) Go to Proxies

5. Uncheck Web Proxy (HTTP) and Secure Web Proxy (HTTPS) to disable the HTTP and HTTPS proxies

6) Click OK, then Apply.
