### **🚀 New**

* You can now move your Header Editor rules to Requestly with one click, seamless migration made easy. [Learn more](/general/imports/header-editor)

### **✨ Improvements**

* Added validation to block invalid header names in API requests.

### **🐛 Fixes**

* API Client now displays the complete set of response headers. Previously, only a limited few were shown.

* Fixed scroll issues in API Client request tabs for smoother navigation.

* Fixed missing or incorrect icons in Rule and Mock editors.

* Fixed a bug that was adding duplicate listeners in the background.
