### 💎 Improvements

1. **New Look for Desktop App on Mac** • <a target="_blank" href="https://github.com/requestly/requestly-desktop-app/releases/tag/v1.8.0">v1.8.0</a>

   The macOS desktop app now opens in a frameless window, providing a cleaner and more modern interface.

2. **Improved Scripts in API Client**\
   Refactored scripts feature in the API Client to optimize performance and reduce bugs.

### 🐞 Fixes

1. Resolved an issue where the undo operation (Cmd+Z) was not functioning correctly in the API Client.

2. Fixed incorrect data clearance warnings that appeared when changing resource or response types in rule editors, even when inputs were empty.
