### 🚀 New
- Added support for **[SOAP APIs](https://docs.requestly.com/general/api-client/send-api-request/send-soap)**. (#4519)

### 🐛 Fixes
- Fixed an issue where **global environment variables could not be created in local workspaces**. [#4533](https://github.com/requestly/requestly/pull/4533)  
- Fixed **OpenAPI import issues in local workspaces** to ensure smoother imports. [#4528](https://github.com/requestly/requestly/pull/4528)  
- Added safeguards to **prevent crashes caused by cyclic parent references in collections**. [#4517](https://github.com/requestly/requestly/pull/4517)
