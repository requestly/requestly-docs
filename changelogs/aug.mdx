### **🚀 New**

* Multipart Form Upload Support: You can now upload files using multipart/form-data, making API testing more flexible and realistic.

### **🛠️ Improvements**

* Faster Load Times: OS detection results are now cached to improve performance during startup.

### **🐛 Fixes**

* Updated Blocklist: Default blocklist in storage is now consistent across sessions.

* Rule Import Feedback: Success message now correctly appears when importing rules in Automation.
