You can upload and use JavaScript and CSS files on Requestly’s File Server. This lets developers host JS or CSS files for testing and development purposes. Here are the steps to create a JS/CSS File:

<Steps>
  <Step title="Go to the File Server in Requestly.">
    From the Requestly dashboard, click on "Files" then “JS/CSS Files” in the left menu bar.
  </Step>

  <Step title="Click on &#x22;New file&#x22; to create a empty file or “Upload file” to upload a JS or CSS file directly.">
    <img src="/images/create-new-mock-file/9bcacb65-d07b-40cd-a0a7-436d722ab5b8.png" align="left" fullwidth="false" />
  </Step>

  <Step title="Select the type of file you want to host.">
    From the popup, select the type of file you want to host (CSS or JS)

    <img src="/images/create-new-mock-file/567d09da-9b00-4b35-b1bb-2a505c18a6db.png" align="left" fullwidth="false" />
  </Step>

  <Step title="Configure the file">
    <img src="/images/create-new-mock-file/79c6670c-e2de-4d49-b5f7-a7b79ca4b0a5.png" align="left" fullwidth="false" />

    Enter the following details:

    * **Name**: Provide a descriptive name for the file.

    * **Data**: Input the file content directly or upload the file.

    * **Endpoint**: Define a unique endpoint for accessing the file.
  </Step>

  <Step title="Click on &#x22;Create&#x22; to finalize the hosted file.">
    <img src="/images/create-new-mock-file/c7aceb4a-9db0-433d-90e0-50544dec81d5.png" align="left" fullwidth="false" />

    Once created, you can copy the generated URL using “Copy URL” button to access the hosted file and see it in action.
  </Step>
</Steps>
