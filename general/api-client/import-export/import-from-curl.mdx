Requestlyâ€™s API Client allows you to easily **import cURL commands** and convert them into fully editable API requests. This makes it simple to migrate from terminal-based workflows or share API requests across tools.

## How to Import a cURL Command

<Steps>
  <Step title="Download and Open Requestly Dashboard">
    Download and launch the Requestly API Client (available as a web app or desktop app). Navigate to the **API** section from the dashboard.

    <img src="/images/import-from-curl/f4daa1cd-7634-4b8b-82a0-959baa0baceb.png" align="center" fullwidth="false" />
  </Step>

  <Step title="Click on &#x22;Import&#x22; and Select cURL">
    In the top-left corner of the API Client, click the **Import** button. From the dropdown, choose **cURL**.

    <img src="/images/import-from-curl/bc62d7c2-3620-4dba-81c1-4587859dcac0.png" align="center" fullwidth="false" />
  </Step>

  <Step title="Paste Your cURL Command">
    A popup will appear. Paste your raw cURL command into the input box.

    <img src="/images/import-from-curl/8d2b68a2-974a-47a9-b282-3c634893706c.png" align="center" fullwidth="false" />
  </Step>

  <Step title="Click &#x22;Import&#x22;">
    Click the **Import** button, and Requestly will convert your cURL into an editable API request, ready to use in your workspace.

    <img src="/images/import-from-curl/e2d5d27b-91d9-488e-ba7e-c9ac1faf4eca.png" align="center" fullwidth="false" />
  </Step>
</Steps>
